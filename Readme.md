## AI Task Planner Agent

### Overview
This project demonstrates a simple agentic AI system that converts natural language instructions into structured JSON decisions and executes the appropriate tools.

### How It Works
1. Takes user input in natural language
2. Extracts values using regex
3. Detects user intent (operation)
4. Generates JSON-based decisions
5. Executes tools and saves output

### Key Concepts Used
- Agentic decision-making
- Tool-based execution
- JSON as an action interface
- Input validation and error handling

### Example

**Input**
multiply 20 and 30

**Output**
600
